# SQL-case-study
This project focuses on analyzing customer data using PostgreSQL. It includes data cleaning, transformation, and querying to extract meaningful insights such as customer income distribution, demographics, and behavioral patterns.
The dataset contains 5000+ customer records with attributes like income, education, marital status, children, and home ownership.
🔹 Objectives
Perform data cleaning and preprocessing
Handle datatype issues (e.g., money to numeric)
Analyze customer demographics
Generate business insights using SQL queries
Practice real-world SQL scenarios
🔹 Dataset Information

The dataset includes the following columns:

Customer ID
Name
Annual Income
Education Level
Marital Status
Number of Children
Home Ownership
Occupation
Birth Date
🔹 Key SQL Concepts Used
SELECT, WHERE, ORDER BY
Aggregate Functions (AVG, SUM, COUNT)
GROUP BY & HAVING
Subqueries
Data Type Casting (money → numeric)
Filtering using LIKE operator
